Контрольні питання

1. Яка структура HTML-документу?
Зазвичай HTML-документ складається з основних тегів html, head, title, body

2. Дати визначення тега.
Тег - це елемент HTML, який використовується для структурування вмісту веб-сторінки.

3. Які теги використовуються при роботі з таблицями?
table - оголошення таблиці, tr - оголошення рядка таблиці, th - оголошення заголовка стовпця/рядка таблиці, td - для оголошення комірки таблиці tr - для оголошення рядка таблиці

4. Які теги використовуються при роботі з зображеннями?
img - для відображення зображення на сторінці

5. Які теги використовуються при роботі зі списками?
ul - для невпорядкованого списку, ol - для впорядкованого списку, li - для елемента списку

6. Перерахувати засоби опису таблиць в html.
Атрибути caption - для опису, border - для встановлення ширини межі таблиці, cosplan - об'єднує декілька комірок по горизонталі,
rowspan - об'єднує декілька комірок по вертикалі
